# StudentRegistrationApp.java
Java Swing desktop application for new student Registration with sqlite backend and Csv storage
 Student Registration Application

 ## Description
This is a Java desktop application developed using NetBeans.
The application allows users to register students by entering
their details through a graphical user interface (GUI).
The data is stored persistently in a local database.

 ## Technologies Used
- Java (Swing)
- NetBeans IDE
- SQLite Database

 ## Database
The application uses an SQLite database file named `student.db`.
The database file is automatically created when the application
is run and is used to store student registration records.

## How to Run the Application
1. Clone or download the repository.
2. Open the project in NetBeans.
3. Run the project.
4. Fill in the student registration form and submit the details.

## Output
Submitted student records are saved in the database and can be
retrieved or displayed by the application.
